# HireLens AI

AI-Powered ATS Resume Analyzer & Interview Assistant

## Features

- Resume Upload (PDF/DOCX)
- ATS Score Calculation
- Skills Extraction
- Matched & Missing Skills Detection
- AI-Powered Resume Feedback (Groq LLM)
- Modern Dashboard UI

## Tech Stack

- Python
- Streamlit
- NLP
- Groq API
- PDF Processing

## Installation

pip install -r requirements.txt

Create a .env file:

GROQ_API_KEY=YOUR_API_KEY

Run:

streamlit run app.py
